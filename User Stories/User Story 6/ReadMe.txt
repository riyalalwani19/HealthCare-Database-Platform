Title - User Story 6

Synopsis - This file will help a user to run user story 6 in his enviornment.

User Story -As a NGO we want to know percentage share of diseases in the database that have doctors available to cure them in the city so that they can submit a report to the government. In addition to that they also want to know percentage share of speciality of doctors that are available in the city for the dieases.

Prerequisites - Following are the database and code related prequisites that need to be fullfilled before running the script.

1. Database - 	a. MongoDB server should be up and running.
				b. Data should have been inserted in all databases.
	
2. Code - 	a. Install python packages (matplotlib and pymongo)
