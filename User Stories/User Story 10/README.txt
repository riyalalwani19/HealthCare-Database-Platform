Title - User Story 10

Synopsis - This document is the guide to run User Story 10 in their environment

User Story - An advertisement company wants to set up an advertisement of a drug on our website and wants to know the best 8 hours to set up that advertisement, so they get the maximum unique audience possible

Prerequisites - Following are the database and code related prequisites that need to be fullfilled before running the script.

1. Database - 	a. Redis server should be up and running.
		b. Data should have been inserted in Redis database.

2. Code - 	c. Install python packages (redis, datetime, pandas, matplotlib, re)


Inputs - Program does not ask for any input. This will represent best 8 hours in graph
